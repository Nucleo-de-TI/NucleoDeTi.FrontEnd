Objetivos do GENTI
  • Criar parcerias com Empresas de Sorocaba e Região
  • Auxiliar a Universidade no desenvolvimento tecnológico sempre que necessário;
  • Engajar os alunos como parte do desenvolvimento de uma integração com a Universidade e Empresas;
  • Desenvolver eventos, treinamentos e disponibilizar para as empresas alunos qualificados para o mercado de trabalho;
  • Criar e manter uma estrutura para os cursos como servidor e uma página própria para uso dos projetos desenvolvidos pelo GENTI.
